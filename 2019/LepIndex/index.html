TaxonWorks?

A Virtual Research Environment (VRE) for biodiversity scientists

Who? Individuals, labs, small-medium sized teams

Emphasis on solid, extensible, and flexible semantics reflecting researchers workflows

Workflows from inception to publication, from specimens up to (not including) genomic level data

TaxonWorks - Production highlights

3i World Auchenorrhyncha Database - 115k+ names; 234k observations, >1my rows of metadata

Universal Chalcidoidea Database - 86k+ names + >1.2my rows of metadata

INHS, Frost, NCSU Insect collections - > 1my specimens, 300k+ collecting events

LepIndex - 304288K names, 600k+ images, etc.

Several "small" efforts, 2-5k names

Coming - Species Files - Orthoptera, Plecoptera etc. well over 150k more names
